# Cardiovascular Disease Prediction

## Project Overview
This project is developed as part of the Corizo Generative AI Mini Project. It focuses on predicting the possibility of cardiovascular disease using machine learning techniques and patient health data.

## Objective
The objective of this project is to analyze health-related factors and build a predictive model that helps identify the risk of cardiovascular disease.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
The project uses the *cardio_train.csv* dataset, which contains patient health information used for training and testing the model.

## Project Files
- Gen AI mini project.ipynb – Jupyter Notebook containing the complete implementation.
- cardio_train.csv – Dataset used in the project.

## Output
The notebook includes the complete code along with the generated outputs for each step.
